# Task Manager 

**Консольное приложение на C#**

## Описание проекта

Данный проект представляет собой консольное приложение «Менеджер задач», реализованное на языке C# с использованием платформы .NET 8.0.
Приложение позволяет управлять задачами и проектами, работать с дедлайнами, зависимостями, визуализацией данных и уведомлениями.


## Структура решения

Решение содержит **четыре проекта**:

```
Project_4.sln
│
├── Project_4          
├── Library            
│
├── Project_4_dop     
└── Library_dop       
```


## Назначение проектов

### Project_4 

Базовая версия консольного приложения «Менеджер задач».

Реализует:

* работу с задачами
* консольное меню
* ввод и вывод данных
* цикл повторного выполнения
* сохранение данных в файл


### Library 

Библиотека классов для базовой версии.

Содержит:

* модель задачи
* перечисления статусов и приоритетов
* методы чтения и записи задач в файл
* базовую бизнес-логику


### Project_4_dop 

Расширенная версия приложения.

Реализует:

* визуализацию через Spectre.Console
* работу с проектами
* дедлайны и зависимости
* асинхронную обработку
* Telegram-уведомления
* прогресс задач


### Library_dop

Расширенная библиотека классов.

Содержит:

* расширенные модели `Task` и `Project`
* зависимости задач
* работу с датами
* импорт / экспорт CSV и JSON
* асинхронную работу с файлами
* фоновые сервисы


## Используемые технологии и библиотеки

### Язык и платформа

* **C#**
* **.NET 8.0**
* Консольное приложение


### Spectre.Console

Используется в проекте **Project_4_dop** для:

* табличного отображения задач
* интерактивных меню
* визуального выделения просроченных задач
* отображения прогресса задач (ProgressBar)
* форматированного и цветного вывода

Пакет:

```xml

```


### Newtonsoft.Json

Используется для:

* экспорта задач и проектов в JSON
* импорта данных
* сериализации расширяемых моделей

Пакет:

```xml
Newtonsoft.Json (13.0.3)
```

### Telegram.Bot

Используется для:

* отправки уведомлений о приближающихся дедлайнах
* информирования о просроченных задачах
* асинхронной фоновой работы бота

Пакет:


### Асинхронность

* `async / await`

## Работа с данными

### Хранение задач

* При запуске приложения запрашивается путь к файлу
* Проверяется корректность пути
* Обрабатываются ошибки ввода
* Поддерживаются форматы:

  * TXT
  * CSV
  * JSON

### Импорт / Экспорт

* Экспорт задач:

  * CSV
  * JSON
* Импорт задач из этих форматов
* При добавлении новых полей они автоматически учитываются

## Реализованный функционал

### Базовая часть

* просмотр всех задач
* добавление новой задачи
* удаление задачи
* изменение статуса
* автоматическая генерация ID
* цикл повторной работы
* сохранение данных при выходе
* обработка ошибок ввода


### Расширенная часть

#### Визуализация

* таблицы задач (Spectre.Console)
* интерактивная фильтрация и сортировка

#### Работа с датами

* дата создания
* дата изменения
* сортировка и фильтрация по датам

#### Зависимости задач

* поддержка множественных зависимостей
* защита от циклических зависимостей
* блокировка задач при невыполненных условиях

#### Дедлайны

* поле Deadline
* визуальное выделение просроченных задач
* предупреждения о приближении срока

#### Асинхронность

* асинхронная работа с файлами
* фоновые проверки дедлайнов


### Дополнительные задачи

#### Управление проектами

* создание, удаление и переименование проектов
* привязка задач к проектам
* отображение задач по проектам
* статистика выполнения

#### Telegram-уведомления

* уведомления о дедлайнах
* фоновая отправка сообщений

#### Прогресс задач

* процент выполнения задачи
* визуальный progress-bar

1. Открыть `Project_4.sln`
2. Выбрать нужный стартовый проект:

   * `Project_4_dop` — расширенная версия
3. Запустить приложение
4. Следовать инструкциям в консоли
